Sobre este Website

Uma aplicação web Django com arquitetura profissional em produção.

🐍 Backend
  • Python 3.12 + Django 5.2
  • Django Ninja (API RESTful)
  • Gunicorn (WSGI)
  • PostgreSQL (Neon)
🎨 Frontend
  • Bootstrap 5.3
  • Django Templates
  • Whitenoise (static)
  • Cloudinary (media)
☁️ Infraestrutura
  • Docker (container)
  • GitHub Actions (CI/CD)
  • Servidor DEISI
  • Neon + Cloudinary

🥊 API RESTful de Boxe

A aplicação inclui uma API RESTful construída com Django Ninja, protegida por API Key. Permite gerir lutadores, combates e títulos de boxe com operações CRUD completas.

Documentação Swagger ↗ Combates Open Boxing

🔄 Pipeline CI/CD

  1. git push → código sobe para o repositório GitHub na organização pw-2425-alunos
  2. GitHub Actions → workflow build-push-deploy.yml dispara automaticamente
  3. Build Docker → imagem construída com o Dockerfile
  4. Deploy → container enviado para o servidor DEISI
  5. Live → aplicação disponível em produção
